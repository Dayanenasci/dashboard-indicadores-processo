# Dashboard de Indicadores de Processo

Projeto desenvolvido para simular um painel de acompanhamento de processos internos, com foco em governança, controle e tomada de decisão.

## Objetivo

Acompanhar indicadores de desempenho do processo de atendimento, garantindo visibilidade, rastreabilidade e melhoria contínua.

## Indicadores Monitorados

- Tempo médio de atendimento
- Percentual de chamados dentro do SLA
- Volume de chamados por prioridade
- Chamados por status

## Tecnologias (simuladas)

- Dados estruturados em CSV
- Modelagem para indicadores
- Visualização em Power BI
- Automação com Power Automate (conceitual)# dashboard-indicadores-processo
