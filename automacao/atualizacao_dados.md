# Automação de Atualização dos Dados

## Objetivo
Automatizar a atualização da base de dados utilizada no dashboard.

## Fluxo Proposto
- Gatilho: novo chamado criado ou atualizado
- Atualização automática da base
- Refresh do dashboard
- Notificação em caso de chamado fora do SLA

## Ferramenta
- Power Automate
